## 도메인 지식 병목을 AI가 어떻게 흡수했는지
- 계약체결비용/계약유지비용은 보험업 실무자가 아니면 "왜 두 계정을 굳이 나눠서 배분해야 하는가"를 판단하기 어려운 항목인데, insurance_accounting_guide.md의 샘플 도메인 지식(§1·§2)을 통해 AI가 신계약비 이연상각 회계기준과 연동된 구분 논리를 근거로 명시하며 판단.

## Phase 0 사전 스캔의 효과
- 공통 계정 4건(임차료·소모품비·통신비·여비교통비)이 부서마다 서로 다른 드라이버로 배분되는 일관성 문제를 사전에 방지 — Phase 0 스캔이 이 계정들을 "구분: 공통"으로 미리 표시해 두었기 때문에, account-classifier와 driver-recommender가 시작부터 "이미 확정된 값이 있는지"를 확인하고 재사용하는 절차를 따름.
